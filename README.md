# 🤝 Let's Connect – Speeddating Timer

Ein schlichter, beamer-tauglicher Timer für moderierte Speeddating- und Kennenlernrunden. Keine Installation, keine Abhängigkeiten – einfach die HTML-Datei im Browser öffnen.

https://kerk.github.io/timer/

---

## ✨ Features

- **Diskussionstimer** mit großem Kreisring-Countdown (gold → orange → rot)
- **Partner-Pause** zwischen den Runden mit eigenem Countdown und Vorschau der nächsten Frage
- **Töne** bei 30 Sekunden verbleibend und wenn der Timer abläuft
- **Fragen verwalten** – beliebig viele Fragen eingeben, per Drag & Drop sortieren
- **Zeiten konfigurierbar** – separate Einstellung für Diskussion und Partnerwechsel-Pause
- **Automatische Speicherung** aller Fragen und Einstellungen im Browser (localStorage)
- **Vollbild-optimiert** – skaliert auf jedem Beamer und Bildschirm
- **Tastaturkürzel** für einfache Steuerung vom Laptop aus

---

## 🚀 Schnellstart

1. `index.html` herunterladen
2. In einem modernen Browser öffnen (Chrome, Firefox, Edge, Safari)
3. Fragen eingeben, Zeiten einstellen, auf **Starten** klicken

Keine Internetverbindung erforderlich (außer für die Schriftarten beim ersten Laden).

---

## ⌨️ Tastaturkürzel

| Taste | Aktion |
|-------|--------|
| `Leertaste` | Pause / Weiter |
| `→` | Nächste Frage überspringen |
| `R` | Aktuelle Runde neu starten |

---

## 🖥️ Ablauf einer Session

```
[Setup] → [Diskussion ⏱] → [Partner finden 🤝] → [Diskussion ⏱] → ... → [Fertig 🎉]
```

Nach jeder Diskussionsrunde erscheint automatisch der Partner-Pause-Screen. Sobald die Pause abläuft (oder manuell übersprungen wird), startet die nächste Runde.
